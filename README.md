# Ogigia Escape
A game I made for the ThessGameJam organised by ACM. The only third party library used is Raylib. All assets were created by me.

## How to run
You can download the executable in the tags both for Windows and for Linux; 64 bit of course.

## How to build (if you don't trust the executables)
Clone the repository, go into the new directory and run the following commands:
```
make deps release
./build/release/ogigia
```
or for a windows build:
```
make windows
build/release/ogigia.exe
```
